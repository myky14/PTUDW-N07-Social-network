# ĐỒ ÁN KẾT THÚC MÔN - PHÁT TRIỂN ỨNG DỤNG WEB
## Đề tài: Mạng xã hội - Nhóm 07

### Tên đề tài: PetConnect - Mạng xã hội dành cho người yêu thú cưng

### Mô tả đề tài
PetConnect là một website mạng xã hội dành cho người yêu thú cưng, cho phép người dùng đăng bài viết, chia sẻ hình ảnh thú cưng, tương tác thông qua lượt thích và bình luận, cũng như kết nối với cộng đồng có cùng sở thích. Hệ thống được xây dựng bằng Bootstrap cho giao diện, lập trình hướng đối tượng (OOP) theo mô hình MVC ở Backend và tích hợp AJAX để xử lý dữ liệu động, mang lại trải nghiệm người dùng tốt hơn.
Xây dựng một nền tảng mạng xã hội cơ bản cho phép người dùng đăng ký, kết bạn, chia sẻ bài viết và tương tác qua lại, kết nối thú cưng. Hệ thống sử dụng PHP (OOP + MVC), Bootstrap và AJAX.
